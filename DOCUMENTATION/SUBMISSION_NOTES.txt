SETUP INSTRUCTIONS
================================================================================

REQUIREMENTS:
- Node.js 14.0 or higher
- npm (comes with Node.js)

INSTALLATION:
1. Extract this folder to your desired location
2. Open terminal/command prompt in the project directory
3. Run: npm install
4. Run: node app.js
5. Open browser and navigate to: http://localhost:3000

TO STOP THE SERVER:
- Press Ctrl+C in the terminal

================================================================================
TROUBLESHOOTING
================================================================================

Q: Port 3000 already in use
A: Run on a different port: PORT=3001 node app.js

Q: Module not found error
A: Run: npm install (ensure node_modules installed)

Q: CSV file not found
A: Verify schools_database.csv is in the project root directory

Q: Cannot connect to localhost:3000
A: Check that the server started successfully (look for port 3000 message)
